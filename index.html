Projects I've worked on

Spell Checker

Developed an advanced text processing tool that utilizes various data structures and algorithms to tokenize input text into words for spell checking. This tool employs the Levenshtein distance algorithm to accurately suggest alternatives for misspelled words. Featuring a command-line interface, it allows users to efficiently input text, receive suggestions, and correct errors in large text documents.

Technologies: Python, Levenshtein Distance, Command-line Interface

ShowSeeker

Created a web application for discovering and purchasing tickets for local events and concerts. The platform features an intuitive interface for browsing events, buying tickets, and organizing schedules. It ensures user security and data integrity through robust backend functionalities, facilitating user registration and verifying credentials. With streamlined payment processing and reliable performance, the application provides a seamless and engaging user experience.

Technologies: Python, HTML, CSS, Javascript, Flask, Git
